# Adaptive Cruise Control (ACC)

## Objective
Develop a basic Adaptive Cruise Control system using Simulink and Stateflow.

## Description
This project models a vehicle following scenario where the ego vehicle adjusts its speed to maintain a safe distance from the lead vehicle.

## Features
- Vehicle dynamics model
- Distance control logic
- Speed control system
- Scenario simulation

## Tools
- MATLAB
- Simulink
- Stateflow

## Structure
- model/ → Simulink models
- scripts/ → MATLAB scripts
- images/ → screenshots of model and results
- docs/ → notes or reports

Work in progress
